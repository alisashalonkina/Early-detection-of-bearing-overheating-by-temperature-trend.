#
Оценка устойчивости модели диагностики к изменению условий эксплуатации. 
Партнёр: НИЛ Технической самодиагностики и самоконтроля приборов и систем  ЮУрГУ (ПАО «Северсталь»); 
Тип данных: Временные ряды
Исходные данные: Вибрационные сигналы подшипников из набора CWRU (Case Western Reserve University) при различных нагрузках и скоростях. 
Классы: нормальное состояние, дефект внутреннего кольца, внешнего кольца, тел качения. 
https://engineering.case.edu/bearingdatacenter/download-data-file
https://data.nasa.gov/dataset/ims-bearings
